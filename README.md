# 🌍 AI Travel Planner

An AI-powered travel planner that generates personalized trip plans based on budget, duration, and destination using machine learning and curated travel data.

---

## 🚀 Features
- 30+ destinations supported  
- Smart travel style recommendation  
- Dynamic budget breakdown  
- Day-wise itinerary generation  
- Clean and user-friendly Streamlit UI  

---

## ⚙️ How It Works

- User selects destination, budget, and number of days  
- ML model predicts travel preferences  
- Destination dataset provides insights (cost, safety, best months)  
- App generates activities and day-wise itinerary dynamically  

---

## 🛠 Tech Stack
- Python  
- Streamlit  
- Scikit-learn  
- Pandas
